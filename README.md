# 시간복잡도

## ✨입력값과 문제를 해결하는 데 걸리는 시간과의 상관관계

"얼마나 오래 걸리는가?"

빅오 표기법 [O(n)]

코드를 작성할 때, 나의 코드가 얼마나 효율적인지를 비교할 때 사용된다.

예를 들어, 중첩 for문을 사용했을 때 효율은 비약적으로 나빠진다.

## 이중 for문일 경우

O(n) * O(n) = O(n²) 가 되는데  
입력값이 작을 때는 성능 차이가 크지 않지만,  
1000개의 인덱스를 가진 배열이 입력될 경우, 총 1,000,000번의 연산이 수행되어 성능이 급격히 저하된다.
